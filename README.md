# Startup-Funding-Analysis
This project provides a comprehensive audit of the Indian startup ecosystem. I transformed a raw dataset of over 3,000 messy entries into an interactive, AI-driven dashboard that identifies high-growth sectors, funding "winters," and the strategic shift from Equity to Debt financing.

#Raw financial data is rarely "clean." This dataset was plagued by encoding errors, inconsistent naming, and "hidden" characters. I used Python (Pandas) to perform advanced data engineering:

#Built a keyword-based mapping function to consolidate 500+ messy "Sub-Verticals" into 6 high-level Service Categories.

#Financial Modeling: Grouped 100+ unique "Investment Types" into a strategic 3-Bucket System:
#Early Stage (Seed, Angel, Series A)
#Growth Stage (Series B, C)
#Late Stage (Series D+, Private Equity)

Through the Power BI Dashboard, I uncovered three critical market trends
#The Rise of Venture Debt: Analysis showed a 16x increase in Debt/Loan funding between 2017 and 2019, suggesting a shift in founder strategy toward non-dilutive capital.
#Bangalore as the "Scaling Capital": AI-powered Key Influencers analysis proved that being in the "Growth Stage" increases the likelihood of a startup being in Bangalore by 2.32x.
#The "Big 3" Dominance: Mumbai, Bangalore, and NCR/Gurgaon capture over 80% of total venture capital, though Tier-2 cities like Bhubaneswar are emerging as high-efficiency hubs.

Language: Python 3.x
#Libraries: Pandas (Data Modeling), NumPy (Vectorized Operations), Matplotlib (Initial EDA).
